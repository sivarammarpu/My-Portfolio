# Sivin Portfolio

A modern, high-performance portfolio website built with Next.js, TypeScript, and Tailwind CSS.

Live-Link = https://my-portfolio-3iv7.vercel.app/

## Features

- **Modern UI/UX**: Clean, dark-themed design with glassmorphism and subtle gradients.
- **Advanced Animations**:
  - **Magnetic Buttons**: Interactive buttons that follow your cursor.
  - **Spotlight Cards**: Hover effects with dynamic gradient borders.
  - **Parallax Background**: Background elements that move with mouse interaction.
  - **3D Tilt**: Project cards that tilt in 3D space.
  - **Typewriter Effect**: Dynamic text animation in the Hero section.
  - **Scroll Progress**: Visual indicator of reading progress.
- **Responsive Design**: Fully optimized for all device sizes.
- **Performance Focused**: Built with Next.js for optimal speed and SEO.

## Tech Stack

- **Framework**: [Next.js 14](https://nextjs.org/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)

## Getting Started

1. Clone the repository:

    ```bash
    git clone https://github.com/sivarammarpu/My-Portfolio.git
    ```

2. Install dependencies:

    ```bash
    npm install
    ```

3. Run the development server:

    ```bash
    npm run dev
    ```

4. Open [http://localhost:3000](http://localhost:3000) with your browser.

## Deployment

This project is optimized for deployment on [Vercel](https://vercel.com/).

1. Push your code to a GitHub repository.
2. Import the project in Vercel.
3. Deploy!
